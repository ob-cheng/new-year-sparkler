## Historia

Stworzyłem ten projekt dla mojego młodszego brata, który uwielbia noworoczne zimne ognie. Początkowo myśleliśmy, że w sklepach wszystko zostało wyprzedane, więc chciałem mieć pewność, że i tak będzie mógł się nimi cieszyć cyfrowo.

"Latarnik" w tle jest inspirowany prawdziwą tradycją w **Brześciu na Białorusi** (gdzie się urodził), gdzie lokalny latarnik każdego wieczoru ręcznie zapala lampy naftowe na ulicy Sowieckiej.

Na szczęście w końcu udało nam się znaleźć prawdziwe ognie! Ale ta mała strona wyszła tak ładnie, że chciałem podzielić się nią ze światem.

## Funkcje

### Interaktywny Zimny Ogień
- Kliknij, aby zapalić, przeciągnij, aby machać
- Realistyczne efekty cząsteczkowe z fizyką bezwładności
- Kolory oparte na temperaturze: iskry przechodzą z białego gorąca → żółty → pomarańczowy → czerwony

### Atmosferyczne Tło
- Proceduralna postać "Latarnika", która chodzi i zapala zabytkowe latarnie uliczne
- **Trolluj Latarnika:** Kliknij dowolną latarnię, aby ją zgasić. Jest on bardzo cierpliwym profesjonalistą i za każdym razem spokojnie wróci, aby ją ponownie zapalić. Jak długo zdołasz go zająć?
- Migoczące gwiazdy i gradientowe niebo

### Tryb Śniegu
- Przełącz efekt padającego śniegu

### Tryb AR (Eksperymentalny)
- Użyj kamery internetowej do śledzenia dłoni (obsługiwane przez MediaPipe)
- Gest kciuka w górę, aby zapalić zimny ogień
- Trzymaj zimny ogień w swojej prawdziwej dłoni

### Obsługa Wielojęzyczna
- Angielski, Rosyjski (Зажигай!), Polski (Zapal!)

## Uruchamianie Lokalne

**Wymagania:** Node.js (v14+)

```bash
# Klonowanie i instalacja
git clone https://github.com/ob-cheng/new-year-sparkler.git
cd new-year-sparkler
npm install

# Uruchomienie serwera deweloperskiego
npm run dev
```

Odwiedź `http://localhost:5173/new-year-sparkler/`

### Dostępne Skrypty

| Komenda | Opis |
|---------|-------------|
| `npm run dev` | Uruchom serwer deweloperski |
| `npm run build` | Buduj do produkcji |
| `npm run preview` | Podgląd buildu produkcyjnego |
| `npm run deploy` | Wdróż na GitHub Pages |
