## История

Я создал этот проект для своего младшего брата, который просто обожает бенгальские огни. Мы думали, что всё раскупили в магазинах, поэтому я хотел убедиться, что он сможет насладиться ими хотя бы в цифре.

"Фонарщик" на заднем плане вдохновлен реальной традицией в **Бресте, Беларусь** (где он родился), где настоящий фонарщик вручную зажигает керосиновые фонари на улице Советской каждый вечер.

К счастью, мы всё-таки нашли настоящие огни! Но этот маленький сайт получился таким уютным, что я решил поделиться им со всем миром.

## Особенности

### Интерактивный Бенгальский Огонь
- Нажми, чтобы зажечь, тяни, чтобы махать
- Реалистичные частицы с инерцией и наклоном
- Тепловая карта цветов: искры меняются от раскаленного белого → желтый → оранжевый → красный

### Атмосферный Фон
- Процедурный "Фонарщик", который ходит и зажигает старинные уличные фонари
- **Троллинг Фонарщика:** Нажми на любой фонарь, чтобы погасить его. Он очень терпеливый профессионал и спокойно вернется, чтобы снова его зажечь. Насколько долго ты сможешь его занимать?
- Мерцающие звезды и градиентное небо

### Режим Снегопада
- Включает и выключает падающий снег

### AR Режим (Экспериментальный)
- Использует веб-камеру для отслеживания рук (на базе MediaPipe)
- Жест "Палец вверх" (Class) чтобы зажечь огонь
- Держи бенгальский огонь в реальной руке

### Поддержка Языков
- Английский, Русский (Зажигай!), Польский (Zapal!)

## Запуск Локально

**Требования:** Node.js (v14+)

```bash
# Клонирование и установка
git clone https://github.com/ob-cheng/new-year-sparkler.git
cd new-year-sparkler
npm install

# Запуск сервера разработки
npm run dev
```

Откройте `http://localhost:5173/new-year-sparkler/`

### Доступные Скрипты

| Команда | Описание |
|---------|-------------|
| `npm run dev` | Запуск сервера разработки |
| `npm run build` | Сборка для продакшена |
| `npm run preview` | Предпросмотр сборки |
| `npm run deploy` | Деплой на GitHub Pages |
